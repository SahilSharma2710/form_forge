# form_forge_generator

Code generator for [form_forge](https://pub.dev/packages/form_forge). This package processes `@FormForge()` annotations and generates `FormController`, `FormWidget`, and `FormData` classes.

**This is the generator package â€” add it as a dev dependency alongside `form_forge`.**

## Installation

```yaml
dependencies:
  form_forge: ^0.1.0

dev_dependencies:
  form_forge_generator: ^0.1.0
  build_runner: ^2.4.0
```

## Usage

See the [form_forge package](https://pub.dev/packages/form_forge) for full documentation and examples.

```bash
dart run build_runner build
```
